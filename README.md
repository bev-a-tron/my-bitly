# This is my bitly service!

## Steps to run

1. Run `bundle install`

1. Create local postgres databases:
    - Login to postgres  
    - `CREATE DATABASE mybitly_development`;
    - `CREATE DATABASE mybitly_test`;

1. Run migrations `rails db:migrate`

1. Run `rails s`

1. Hit `localhost:3000`

## Run specs

1.  Run `rspec`
